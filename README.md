<h1>:books: 미니프로젝트로 진행한 당근 웹사이트 클론코딩</h1>

## 📢 구현한 기능

- 전체화면 기준으로 px값으로 조절을 통해 디자인
- 버튼 활성화를 위해 당근마켓의 버튼 클릭시 이동되는 사이트를 그대로 적용하여 똑같이 구현되도록 적용

## 🛠️ 기술 스택

### Frontend

- **HTML**: 웹사이트의 토대 구현
- **CSS**: 웹사이트의 디자인
- **JavaScript (ES6+)**: 동적 기능 구현

### DevOps

- **Git**: 버전 관리
